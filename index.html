Lenguajes de la web

html

La definición según Wikipedia:


Conocido por sus siglas en inglés de HyperText Markup Language (‘lenguaje de marcas de hipertexto’), HTML es un lenguaje de marcado: un estándar utilizado para la creación de páginas web, que se encarga de definir la estructura básica de éstas por medio de etiquetas. Se puede insertar imágenes, vídeos, audios, texto, juegos, entre otros. Ofrece adaptabilidad, estructuración lógica y es fácil de interpretar tanto por humanos como por máquinas.
Las etiquetas se rodean por corchetes angulares (<,>) y son la estructura básica de HTML. Tienen dos propiedades básicas: atributos y contenido, y cada uno tiene sus diferentes restricciones.
HTML tiene límites, ya que por sí sólo apenas puede describir hasta cierto punto la apariencia del documento, y no puede añadir interactividad a sus elementos. Y para ello necesitamos los siguientes dos lenguajes: CSS y Javascript.

css

La definición según Wikipedia:


Conocido por sus siglas en inglés de Cascading Style Sheets (‘hojas de estilo en cascada’), CSS es un lenguaje de diseño: define y crea la presentación de un documento estructurado escrito con un lenguaje de marcado. Está creado para separar el contenido del documento y la presentación de éste, para mejorar la accesibilidad, la flexibilidad y el control de las características presentacionales. Esto hace posible que un mismo documento se pueda marcar de diferentes estilos para diferentes métodos de renderizado, como de pantalla, impresión y voz. Como también se puede mostrar la misma página de manera diferente dependiendo del tamaño de pantalla o el tipo de dispositivo.
Se determina un esquema de reglas de estilo que se aplican a los elementos según el selector que posea dicho elemento.

javascript

La definición según la W3C:


JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, cada vez que una página web hace algo más que sentarse allí y mostrar información estática para que la veas, muestra oportunas actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo, etc., puedes apostar que probablemente JavaScript está involucrado. Es la tercera capa del pastel de las tecnologías web estándar, dos de las cuales (HTML y CSS) hemos cubierto con mucho más detalle en otras partes del Área de aprendizaje
